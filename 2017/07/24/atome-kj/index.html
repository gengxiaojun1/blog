Atom 编辑器 Markdown书写快捷键

概述

Markdown 的目标是实现「易读易写」。

Markdown 的语法全由一些符号所组成，这些符号经过精挑细选，其作用一目了然。比如：在文字两旁加上星号，看起来就像强调。Markdown 的列表看起来，嗯，就是列表。Markdown 的区块引用看起来就真的像是引用一段文字，就像你曾在电子邮件中见过的那样。

  • 加粗文字 : b

  • 代码: c

  • 斜体: i

  • 图像: img

  • 链接: l

  • 版权声明: legal

  • 填充文字: lorem

  • 引用编号代码格式:ref

  • 外联引用指定编号的链接: rl

  • 外联引用指定编号图片: rimg

  • todo列表 :t

  • 表格： table

注意

请安装autocomplete-plus插件，安装方式请参考 ATOM推荐插件
