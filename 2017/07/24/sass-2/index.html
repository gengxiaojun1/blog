Sass的表达式和控制命令 @for 篇

@for

@for指令常用于循环输出。

@for 有两种使用方式:from start through end和from start to end,
两者的区别在于，前者遍历的范围是 [start, end], 而后者的遍历范围是 [start, end-1]。在@for循环中使用一个固定变量来替代遍历到的元素。如果你想实现从大到小的遍历，只需让 start 大于 end 即可。或者可以这样来理解

through:
